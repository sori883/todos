# ビルドとエラー修正

Rustのビルドおよびclippyエラーを段階的に修正する:

1. ビルド実行: `cargo build 2>&1`

2. エラー出力を解析:
   - ファイルごとにグループ化
   - エラーコード（E0xxx）で分類
   - 重要度順にソート

3. 各エラーに対して:
   - エラーのコンテキストを表示（rustcの提案を含む）
   - 問題を説明
   - 修正を提案
   - 修正を適用
   - `cargo build` を再実行
   - エラーが解決されたことを確認

4. ビルド成功後、clippy チェック:
   - `cargo clippy --all-targets --all-features -- -D warnings`
   - clippy の警告/エラーも同様に修正

5. 以下の場合は停止:
   - 修正が新しいエラーを生む
   - 3回試行しても同じエラーが残る
   - ユーザーが一時停止を要求

6. サマリーを表示:
   - 修正されたエラー
   - 残存エラー
   - 新しく発生したエラー

安全のため、エラーは一つずつ修正すること！
